To deploy an application after making design changes, follow these steps:


1. Go to the assembly.
2. Edit the design. 
3. Remember to commit the changes.
4. Go to the Transition phase.
5. Click **Pull Design.**
6. Verify the changes.
7. Click **Commit & Deploy.**
